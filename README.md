# Odin-recipes

## Proyecto #1 del curso de Odin project
En este repositorio estará condensado la primera parte de lo aprendido durante el curso, veremos una pagina básica construida en HTML que contara con su estructura básica, y los elementos esenciales para la construcción de la prime web

1. Contenido de la página

- Titulo
- Sección principal.
    - sub secciones que contendrán las recetas y los pasos para su realización.

2. Recursos
- `index.html`: Contendrá el la pagina de inicio
- `./img`: Carpeta donde se guardarán las imágenes a utilizar
- `./views`: Carpeta donde se guardarán las diferentes vistas del aplicativo.
